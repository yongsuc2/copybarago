## 이벤트 구조 패턴

### 수집형 이벤트
```
[이벤트 기간]
    ↓
[콘텐츠 진행 시 이벤트 재화 드롭]
    ↓
[이벤트 재화 누적]
    ↓
[이벤트 상점에서 보상 교환]
    ↓
[마일스톤 보상 (누적량 기준)]
```

### 미션형 이벤트
```
[이벤트 기간]
    ↓
[지정된 미션 수행]
  ├ 챕터 N회 클리어
  ├ 장비 N회 강화
  ├ 보물상자 N회 오픈
  └ PvP N회 참여
    ↓
[미션 완료 보상]
    ↓
[전체 미션 완료 시 최종 보상]
```

### 한정 가챠 이벤트
```
[이벤트 기간]
    ↓
[한정 보물상자 출현]
    ↓
[한정 S-Epic 장비/한정 펫 획득 가능]
    ↓
[180회 천장 = 확정 획득]
```
