## 이벤트 구조 패턴

### 수집형 이벤트
```
[이벤트 기간]
    ↓
[콘텐츠 진행 시 이벤트 재화 드롭]
    ↓
[이벤트 재화 누적]
    ↓
[이벤트 상점에서 보상 교환]
    ↓
[마일스톤 보상 (누적량 기준)]
```

### 미션형 이벤트
```
[이벤트 기간]
    ↓
[지정된 미션 수행]
  ├ 챕터 N회 클리어
  ├ 장비 N회 강화
  ├ 보물상자 N회 오픈
  └ PvP N회 참여
    ↓
[미션 완료 보상]
    ↓
[전체 미션 완료 시 최종 보상]
```

### 한정 가챠 이벤트
```
[이벤트 기간]
    ↓
[한정 보물상자 출현]
    ↓
[한정 S-Epic 장비/한정 펫 획득 가능]
    ↓
[180회 천장 = 확정 획득]
```

## AI Agent 이벤트 전략

### 이벤트 우선순위
```
1순위: 신규 유저 이벤트 (7일 카니발) - 무조건 완료
2순위: 한정 가챠 이벤트 - 보석 비축 후 천장 도전
3순위: 수집형 이벤트 - 일일 루틴 중 자연 수집
4순위: 미션형 이벤트 - 일일 루틴과 겹치는 미션 우선
```

### 이벤트 재화 관리
```
IF 이벤트_재화로 교환 가능한 S급_보상 존재 THEN
  해당 보상 우선 교환
ELSE
  성장 재화 (골드/강화석/도면) 교환
NEVER: 이벤트 재화 잔여 상태로 이벤트 종료
```

### 이벤트 일정 대응
```
이벤트 시작 시:
  1. 보상 목록 확인
  2. 필요 재화/미션 수 계산
  3. 일일 목표량 산정
  4. 일일 루틴에 이벤트 동선 추가

이벤트 종료 2일 전:
  IF 목표_미달성 THEN 잔여 보상에 재화 집중 투입
```
