# 카피바라고! 역기획서 - 모험 시스템

## 개요

모험은 게임의 핵심 루프로, 플레이어가 스태미나를 소모하여 챕터에 진입하고,
인카운터를 통해 스킬을 획득하며, 보스를 처치해 챕터를 클리어하는 시스템이다.

---

## 챕터 시스템

### 챕터 진입 조건
- 스태미나 5개 소모
- 항상 다음 미클리어 챕터에 도전 (clearedChapterMax + 1)

### 챕터 구조

모든 챕터는 **60일** 구성이다.

```
[진입] → [인카운터 60턴 반복] → [보스 전투] → [클리어]
```

### 인카운터 유형

| 유형 | 설명 |
|------|------|
| COMBAT | 전투 인카운터 (자동 진행) |
| ANGEL | 천사 - 무료 스킬 선택 또는 회복 |
| DEMON | 악마 - HP 대가로 스킬 |
| CHANCE | 우연 - 보물/회복/축복 중 랜덤 |
| ROULETTE | 룰렛 - 확률 기반 희귀 스킬 |
| LUCKY_MACHINE | 행운 머신 - 확률 기반 골드 도박 |

### 인카운터 상세

#### COMBAT (전투)
- 적과 자동 전투 진행 (선택지 없음)
- 승리 시 다음 일차로 진행, 패배 시 챕터 종료

#### ANGEL (천사)
- 게임 내 설명: "천사가 나타나 스킬이나 회복을 제공합니다."
- 선택지:
  - 스킬 2개 중 1개 선택 (일반/전설 등급, 무료)
  - 체력 30% 회복

#### DEMON (악마)
- 게임 내 설명: "악마가 체력을 대가로 강력한 스킬을 제안합니다."
- 선택지:
  - 최대 체력 20% 소모 → 랜덤 스킬 1개 획득 (모든 등급 가능)
  - 거절 (안전하게 떠나기)

#### CHANCE (우연)
- 게임 내 설명: "행운의 이벤트가 발생했습니다!"
- 3가지 중 랜덤 1개가 선택지로 표시:
  - **상자 열기**: 골드 100~300 획득
  - **치유의 샘**: 최대 체력의 50% 회복
  - **축복**: 골드 200 획득

#### ROULETTE (룰렛)
- 게임 내 설명: "신비한 룰렛이 나타났습니다. 운을 시험해 보세요!"
- 선택지:
  - 일반 돌리기: 80% 확률로 신화 스킬 획득
  - 대박 돌리기: 30% 확률로 불멸 스킬 획득
  - 건너뛰기

#### LUCKY_MACHINE (행운 머신)
- 게임 내 설명: "행운의 머신이 나타났습니다. 위험할수록 보상이 큽니다!"
- 선택지:
  - 안전 머신: 100% 확률, 250골드
  - 일반 머신: 70% 확률, 500골드
  - 위험 머신: 40% 확률, 1000골드

### 인카운터 출현 가중치

| 유형 | 가중치 |
|------|--------|
| COMBAT | 35% |
| ANGEL | 25% |
| DEMON | 10% |
| CHANCE | 20% |
| ROULETTE | 10% |

## 최고 생존 일수

각 챕터별로 플레이어가 도달한 최고 일수를 기록한다.

- 전투 패배 시: 해당 시점의 currentDay 기록
- 보스 패배 시: totalDays 기록 (보스까지 도달)
- 챕터 클리어 시: totalDays + 1 (sentinel 값, 클리어 구분)
- 기존 기록보다 높을 때만 갱신 (Math.max)

## 챕터 보물상자

챕터에서 일정 일수 이상 생존하면 수령할 수 있는 일회성 보상 시스템.

### 마일스톤 정의 (모든 챕터 공통)

| 마일스톤 | 조건 |
|---------|------|
| 15일 | 15일 이상 생존 |
| 25일 | 25일 이상 생존 |
| 40일 | 40일 이상 생존 |
| 챕터 클리어 | 챕터 클리어 |

### 보상 테이블

보상은 챕터 ID에 비례하여 스케일링된다.

| 마일스톤 | 골드 | 보석 | 장비석 | 파워스톤 |
|---------|------|------|--------|---------|
| 15일 | 300×id | 30 | 3 | 0 |
| 25일 | 500×id | 60 | 5 | 1 |
| 40일 | 800×id | 100 | 8 | 2 |
| 클리어 | 1200×id | 150 | 12 | 3 |

### 수령 규칙
- 각 마일스톤은 1회만 수령 가능
- 생존 일수 조건을 만족해야 수령 가능
- 클리어 마일스톤은 해당 챕터 클리어 필요
- 순서 무관하게 조건 충족 시 수령 가능

---

## 탐방 (Travel)

최고 클리어 챕터를 기반으로 골드를 파밍하는 시스템.

### 골드 공식
```
골드 = (스태미나 × 50 × 최고클리어챕터 × 배율) / 100
```

### 배율 옵션
- 3배, 5배, 10배, 20배, 50배
