# 카피바라고! 역기획서 - 전투 시스템

## 전투 방식

| 항목 | 내용 |
|------|------|
| 전투 유형 | 턴제 자동 전투 |
| 선공 | 플레이어 선공 |
| 전투 진행 | 자동 (플레이어 개입 없음) |
| 전투 결과 | 승리 시 보상 획득, 패배 시 챕터 종료 |

## 전투 흐름

```
[전투 시작]
    ↓
[플레이어 턴] → 기본 공격 + 보유 스킬 자동 발동
    ↓
[적 턴] → 적 기본 공격 + 적 스킬 발동
    ↓
[턴 종료] → 턴 기반 효과 처리 (DoT, HoT, 버프/디버프)
    ↓
[승패 판정] → HP 0 이하 시 패배/승리
    ↓
[다음 턴 또는 전투 종료]
```

## 스탯 체계

| 스탯 | 설명 | 강화 경로 |
|------|------|-----------|
| HP | 최대 체력 | 특성, 장비(의상/장신구), 스킬 |
| ATK | 공격력 | 특성, 장비(무기/반지), 스킬 |
| DEF | 방어력 | 특성, 스킬 |
| CRIT | 치명타 확률/배율 | 스킬, 계승(유협) |

## 스킬 발동 구조

### 스킬 등급

| 등급 | 설명 |
|------|------|
| 일반 | 기본적인 추가 공격, 버프, 디버프 |
| 전설 | 마스터리 계열 강화 스킬 |
| 신화 | 극강 효과 (부활, 슈퍼 공격력 등) |
| 불멸 | 대박 룰렛 전용 최상위 스킬 |

### 주요 스킬 유형

| 유형 | 예시 | 효과 |
|------|------|------|
| 직접 공격 | 번개, 광창, 수리검, 검기 | 추가 데미지 |
| 반격 | 반격 계열 | 피격 시 자동 반격 |
| 연타 | 연타 마스터리 | 기본 공격 다회 적중 |
| 치명타 | 치명타 마스터리 | 크리티컬 확률/배율 상승 |
| 흡혈 | 흡혈 계열 | 데미지의 일부를 HP로 회복 |
| DoT | 독소, 화상 | 매 턴 지속 데미지 |
| 버프 | ATK/DEF/HP Proficiency | 스탯 비례 강화 |
| 생존 | 부활(Revive) | 사망 시 1회 부활 |

### S티어 스킬 (최우선 획득 대상)

- HP Proficiency
- ATK Proficiency
- DEF Proficiency
- CRIT Proficiency
- Thunderstorm
- Tyrant
- Revive

## 빌드 시스템 (계승 연계)

| 빌드 | 핵심 스킬 | 계승 루트 |
|------|-----------|-----------|
| 연타/검기 | 연타 마스터리, 검기 | 해골 계승 |
| 반격/광창 | 반격, 광창 | 기사 계승 |
| 치명타/수리검 | 치명타 마스터리, 수리검 | 유협 계승 |
| 스킬/속성 | 속성 스킬, 스킬 강화 | 유령 계승 |

## 전투 난이도 요소

| 요소 | 설명 |
|------|------|
| 적 체력/공격력 | 챕터가 높을수록 증가 |
| 적 스킬 | 고난이도 적은 특수 스킬 보유 |
| 보스 | 챕터 마지막에 등장, 높은 스탯 + 특수 패턴 |
| 엘리트 | 일반 적보다 강화된 중간 보스급 |

## AI Agent 전투 관련 의사결정

### 전투 진입 판단 로직
```
IF 현재HP > 최대HP * 0.5 THEN 전투 진입
IF 흡혈스킬 보유 AND 현재HP > 최대HP * 0.3 THEN 전투 진입
IF 현재HP < 최대HP * 0.3 THEN 회복 우선
```

### 스킬 선택 우선순위 로직
```
1. 현재 빌드(계승 루트)와 시너지가 있는 스킬
2. S티어 범용 스킬 (Proficiency 계열, Revive)
3. 현재 부족한 스탯을 보완하는 스킬
4. 일반 공격 스킬
```

### 전투 결과 예측 요소
- 플레이어 ATK vs 적 HP: 킬 턴 수 추정
- 적 ATK vs 플레이어 HP: 생존 턴 수 추정
- 흡혈량 vs 피해량: 지속 전투 가능 여부
