## 작업 구조

```
Phase 0: 프로젝트 셋업
Phase 1: 도메인 모델 (엔티티 + 값 객체)
Phase 2: 핵심 시스템 (전투 + 챕터)
Phase 3: 성장 시스템 (특성 + 전승 + 장비 + 펫)
Phase 4: 콘텐츠 시스템 (던전 + 타워 + PvP + 탐방)
Phase 5: 경제 시스템 (재화 + 가챠 + 상점)
Phase 6: 메타 시스템 (이벤트 + 세이브 + 일일 루틴)
Phase 7: UI/프레젠테이션 (웹 프로토타입)
Phase 8: 밸런싱 + 테스트
```
