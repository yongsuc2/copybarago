# 카피바라고! 역기획서 - 캐릭터 성장 시스템

## 개요

카피바라고는 다수 영웅 수집이 아닌, **단일 카피바라 캐릭터의 영구 성장** 구조를 채택한다.
성장은 크게 **특성(Talent)** 과 **전승(Heritage)** 두 축으로 나뉜다.

## 특성(Talent) 시스템

### 강화 대상

| 스탯 | 강화 우선순위 | 이유 |
|------|-------------|------|
| ATK | 1순위 | 킬 속도 = 생존력 (빠른 처치가 피해 감소) |
| HP | 2순위 | 생존 턴 수 확보 |
| DEF | 3순위 | ATK/HP 대비 효율 낮음 |

### 레벨당 스탯 증가

| 스탯 | 레벨당 증가 |
|------|-----------|
| HP | +15 |
| ATK | +3 |
| DEF | +2 |

수치: `TalentTable.STAT_PER_LEVEL` 참조

### 강화 비용
- 재화: 골드
- **서브 등급(단) 내에서는 비용 고정**, 다음 서브 등급으로 전환 시에만 비용 상승
- 3개 스탯 모두 동일 비용 (총 레벨 기반)
- 구체적 수치는 `talent.data.json`의 `gradeConfig` 참조

### 진급 경로

6개 메인 등급, 각 등급을 서브 등급(단)으로 세분화:

```
수련생(5단) → 모험가(15단) → 정예(30단) → 달인(50단) → 전사(100단) → 영웅
```

- 총 67개 서브 등급, 각 서브 등급 = 30레벨 (공10 + 방10 + 체10)
- 영웅 도달 총 레벨: 2010
- 등급 경계: 수련생=0, 모험가=60, 정예=210, 달인=510, 전사=1020, 영웅=2010
- UI 표시: "모험가 2단" 형식
- 각 스탯은 서브 등급 내에서 최대 10레벨, 3개 스탯 모두 10 도달 시 서브 등급 진급 + 레벨 리셋
- 비용: 서브 등급 내 고정, `floor(baseCost × costGrowth^(tier-1))`
- 서브 등급 구조: `talent.data.json`의 `gradeConfig`에서 자동 생성

**스탯 보너스**: 모든 서브 등급 전환 시 ATK/DEF 교대 보너스 자동 적용:
- 서브 등급 전환: `tierBonusBase + subTierIdx × tierBonusGrowth` (등급 내에서 점진 증가)
- 메인 등급 전환: `mainGradeBonuses` 대량 보너스
- 구체적 수치는 `talent.data.json`의 `gradeConfig.tierBonusBase/tierBonusGrowth` + `mainGradeBonuses` 참조

**마일스톤 보상**:

- 10레벨 간격 (서브 등급당 2개 마일스톤: +10, +20 위치)
- 서브 등급 경계(30의 배수)를 제외한 간격으로 보상 배치
- 골드 일시 지급과 골드 획득량 % 증가가 교대로 지급
- 구체적 보상은 `talent.data.json`의 `milestoneConfig` 참조
- 도달 시 재능 화면에서 수동 수령
- 이미 지나간 마일스톤도 소급 수령 가능
- 골드 획득량 증가는 챕터 내 전투/인카운터/클리어 골드에만 적용

## 전승(Heritage) 시스템

### 해금 조건
- 특성이 **"영웅" 등급**에 도달 시 해금

### 계승 루트 (5종)

| 계승 | 특화 빌드 | 핵심 전략 | 전용 재화 |
|------|-----------|-----------|-----------|
| 해골 | 연타/검기 | 다회 타격으로 높은 DPS | 뼈 책 |
| 기사 | 반격/광창 | 피격 시 반격으로 안정적 딜 | 기사 책 |
| 유협 | 치명타/수리검 | 크리티컬 기반 폭딜 | 레인저 책 |
| 유령 | 스킬/속성 | 속성 스킬 시너지 극대화 | 영혼 책 |
| 드래곤 | 미출시 | - | - |

### 전승 강화 구조

```
[전용 재화(책) 투입] → [전승 레벨 상승] → [빌드 특화 패시브 강화]
                                              ↓
                                    [해당 빌드 스킬 효과 증폭]
```

### 계승 루트별 시너지 스킬 매핑

#### 해골 계승
- 연타 마스터리 → 공격 시 확률로 추가 타격
- 검기 → 추가 범위 공격

#### 기사 계승
- 반격 → 피격 시 자동 공격
- 광창 → 관통 공격

#### 유협 계승
- 치명타 마스터리 → 크리티컬 확률/배율 증가
- 수리검 → 추가 투사체 공격

#### 유령 계승
- 속성 스킬 전반 → 속성 데미지 강화
- 스킬 강화 계열 → 스킬 발동 효과 증폭

## 성장 자원 흐름도

```
[골드]
  ├→ 특성 강화 (ATK/HP/DEF 레벨업)
  └→ 전승 강화 (계승 루트 레벨업)

[전용 책]
  └→ 전승 강화 (계승별 전용 재화)

[장비 강화석]
  └→ 장비 시스템 (별도 문서)

[펫 사료/알]
  └→ 펫 시스템 (별도 문서)
```

## AI Agent 성장 전략

### 특성 강화 배분 로직
```
골드_배분:
  IF ATK_레벨 < HP_레벨 THEN ATK 강화
  IF HP_레벨 < ATK_레벨 - 5 THEN HP 강화
  DEF는 ATK/HP 대비 절반 비율로 강화
```

### 계승 루트 선택 로직
```
IF 보유_스킬에 연타/검기 다수 THEN 해골 계승
IF 보유_스킬에 반격/광창 다수 THEN 기사 계승
IF 보유_스킬에 치명타 다수 THEN 유협 계승
IF 보유_스킬에 속성 다수 THEN 유령 계승
DEFAULT: 해골 계승 (범용 DPS)
```

### 전승 강화 우선순위
```
1. 선택한 계승 루트의 핵심 패시브 우선 강화
2. 전용 재화(책) 획득 가능 콘텐츠 일일 소화
3. 다른 계승 루트 재화는 비축 (루트 변경 대비)
```
