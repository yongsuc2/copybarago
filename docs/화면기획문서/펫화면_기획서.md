# 펫 화면 기획서

---

## 1. 화면 구성 (위 → 아래)

### 1-1. 화면 헤더
```
펫
```

### 1-2. 재화 정보 카드
```
┌────────────────────────────────┐
│ 펫 알                   {수량} │
│ 펫 먹이                 {수량} │
│                                │
│ [       알 부화       ]        │
└────────────────────────────────┘
```
- 펫 알 0개이면 부화 버튼 비활성

### 1-3. 빈 상태
```
보유한 펫이 없습니다
```
- 펫이 0마리일 때만 표시

### 1-4. 펫 쇼케이스 영역 (상단)
```
┌────────────────────────────────┐
│                                │
│      [PetIcon 확대, 좌우흔들림]  │
│          {펫 이름}              │
│       [{티어}] {등급}           │
│                                │
└────────────────────────────────┘
```
- 선택된 펫을 확대 크기로 표시
- 좌우 흔들림 애니메이션 (idle 모션)
- 펫 이름, 티어, 등급 표시
- 출전 중인 펫은 "출전 중" 뱃지 표시
- 다크 배경 (ColorPalette.Background 계열)

### 1-5. 스탯 카드 (중단)
```
┌────────────────────────────────┐
│ Lv.{N}                        │
│ HP  +{N}    공격력  +{N}       │
│ 방어력  +{N}                   │
│────────────────────────────────│
│ 특수능력  {능력 설명}           │
│────────────────────────────────│
│ 레벨업 시: HP +{N}, ATK +{N}   │  ← 레벨업 프리뷰
│ EXP  ████░░░░ {N}/{N}         │
│ 먹이 {N}개 → 레벨업            │
│────────────────────────────────│
│ [출전] [먹이주기] [⬆Lv.{N}]   │
└────────────────────────────────┘
```
- 현재 레벨, 스탯 보너스 (HP/ATK/DEF)
- 특수능력 설명 (금색 라벨)
- 레벨업 시 능력치 증가 미리보기 (회색 텍스트)
- 경험치 프로그래스 바 + 필요 먹이 수
- 버튼: 출전, 먹이주기, 최대 레벨 일괄 먹이주기

### 1-6. 펫 아이콘 그리드 (하단)
```
┌──┐┌──┐┌──┐┌──┐┌──┐
│🐱││🐉││🦊││🐧││🦁│  ← 5열 CSS Grid
│  ││  ││  ││  ││  │
└──┘└──┘└──┘└──┘└──┘
```
- 5열 그리드
- 셀 배경색 = 펫 등급 색상 (ColorPalette의 등급별 색상 참조)
- 셀 안에 PetIcon 표시
- 선택된 펫: 밝은 테두리 강조
- 출전 중인 펫: 강조 테두리
- 클릭 시 상단 쇼케이스에 해당 펫 표시

## 2. 버튼 상세

### 2-1. 출전 버튼
- 해당 펫을 활성 펫으로 설정
- 이미 출전 중이면 "출전 중" (비활성)

### 2-2. 먹이주기 버튼
- 펫 먹이 1개 소모, 경험치 획득
- 경험치 충분 시 레벨업
- 먹이 0개이면 비활성

### 2-3. 최대 레벨 먹이주기 버튼 (⬆Lv.{N})
- 보유 먹이로 도달 가능한 최대 레벨까지 **필요한 만큼만** 먹이 소모
- 버튼 텍스트: "⬆Lv.{도달레벨}" (예: ⬆Lv.5)
- 현재 레벨이 이미 최대이거나, 먹이가 부족하면 비활성
- 초과 먹이는 소모하지 않음 (정확한 양만 사용)

---

## 3. 경험치/레벨 시스템

- 먹이당 경험치: 데이터 테이블 참조 (Pet 엔티티)
- 다음 레벨 필요 경험치: 데이터 테이블 참조 (레벨에 따라 증가)
- 레벨당 스탯 보너스: 데이터 테이블 참조 (Pet 엔티티)

## 4. 등급 체계

| 등급 | 영문 | 아이콘 배경색 |
|------|------|-------------|
| 일반 | COMMON | ColorPalette 등급색 참조 |
| 희귀 | RARE | ColorPalette 등급색 참조 |
| 에픽 | EPIC | ColorPalette 등급색 참조 |
| 전설 | LEGENDARY | ColorPalette 등급색 참조 |
| 불멸 | IMMORTAL | ColorPalette 등급색 참조 |

## 5. 데이터 매핑

```
펫 목록       → Player.ownedPets
활성 펫       → Player.activePet
펫 알 수량    → Player.resources.get(PET_EGG)
펫 먹이 수량  → Player.resources.get(PET_FOOD)
펫 아이콘     → PetIcon 컴포넌트 (petId, tier)
펫 스탯       → Pet.getGlobalBonus()
경험치 정보   → Pet.exp, Pet.getExpToNextLevel()
특수능력 설명 → PetTable.getAbilityDescription(templateId, grade)
부화 처리     → PetManager.hatchEgg(rng)
먹이 처리     → Pet.feed(amount)
최대레벨 먹이 → PetScreen.feedToMaxLevel() (필요량 계산 후 정확한 양만 소모)
출전 처리     → Player.setActivePet(pet)
```

## 6. 상호작용 흐름

### 알 부화
```
[알 부화] 클릭 → 펫 알 1개 소모 → PetManager.hatchEgg() → 펫 목록에 추가
→ 활성 펫 없으면 자동 출전 → 자동 저장
```

### 먹이주기
```
[먹이주기] 클릭 → 펫 먹이 1개 소모 → Pet.feed(1) → 경험치 획득
→ 경험치 충분 시 레벨업 (스탯 증가) → 자동 저장
```

### 최대 레벨 먹이주기
```
[⬆Lv.N] 클릭 → 필요한 먹이량 계산 → 정확한 양만 소모 → 다회 레벨업
→ 보유 먹이로 도달 가능한 최대 레벨까지 일괄 육성 → 자동 저장
```

### 출전
```
[출전] 클릭 → Player.setActivePet(pet) → 기존 활성 펫 해제 → 자동 저장
```

### 펫 선택 (아이콘 그리드)
```
아이콘 클릭 → 선택 인덱스 변경 → 쇼케이스+스탯카드 해당 펫으로 갱신
```
