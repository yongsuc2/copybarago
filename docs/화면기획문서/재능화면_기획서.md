## 1. 화면 구성 (위 -> 아래)

### 1-1. 화면 헤더
```
재능
```

### 1-2. 등급 정보 카드
```
┌────────────────────────────────┐
│ 등급              모험가 3단    │
│ 서브 등급 진행    12 / 30      │
│ 총 레벨          72 / 210     │
│ 골드 획득량       +4%          │
│ ▓▓▓▓▓▓▓▓░░░░░░░░░ (40%)      │
└────────────────────────────────┘
```
- 등급: `TalentTable.getSubGradeLabel(totalLevel)` — "모험가 3단" 형식
- 서브 등급 진행: `subGradeInfo.levelInTier` / `subGradeInfo.tierLevels` (30 고정)
- 총 레벨: `totalLevel` / `nextGradeThreshold` (다음 메인 등급 경계)
- 골드 획득량: `Player.getGoldMultiplier()` 기반, 0%이면 행 숨김
- 프로그레스 바: 서브 등급 내 진행도 (levelInTier / tierLevels)

### 1-3. 등급 보상 카드 (슬라이딩 윈도우 프로그레스 바)

레퍼런스 이미지처럼 프로그레스 바 위에 보상 아이콘+텍스트가 보이는 구조.
단, 레이아웃은 **가로 방향**.

```
┌────────────────────────────────────────────┐
│ 등급 보상                                   │
│                                             │
│   🪙     ⬆        📈      ⭐       🪙     │
│  720    방+1     +2%    모험가    1.1K     │
│                                             │
│ ▓▓▓▓▓▓▓▓▓▓▓▓░░░░░░░░░░░░░░░░░░░░░░░      │
│ Lv.55                            Lv.70     │
│                                             │
│ 미수령 보상: 2개              [모두 수령]    │
└────────────────────────────────────────────┘
```

#### 슬라이딩 윈도우
- 전체 범위(0~2010)를 한 번에 보여주지 않음
- **15레벨 범위**만 표시 (`VIEW_RANGE = 15`)
- 현재 총 레벨을 **중앙**에 위치시킴 (`totalLevel - floor(VIEW_RANGE/2)` ~ `+VIEW_RANGE`)
- 끝에 도달하면 클램핑 (0 미만 방지, MAX_LEVEL 초과 방지)
- 바 좌우 끝에 `Lv.{viewStart}` ~ `Lv.{viewEnd}` 라벨 표시

#### 노드 종류 (프로그레스 바 위에 표시)

| 노드 | 아이콘 | 표시 내용 | 상태 |
|------|--------|----------|------|
| 골드 마일스톤 | 🪙 | 보상 금액 (K/M/B 축약) | locked / claimable / claimed |
| 골드 획득량 마일스톤 | 📈 | +N% | locked / claimable / claimed |
| 서브 등급 전환 | ⬆ | `{스탯}+{amount}` (예: 방어력+2) | reached / locked |
| 메인 등급 전환 | ⭐ | 등급명 (예: 모험가) | reached / locked |

#### 노드 상태별 스타일

| 상태 | 조건 | 스타일 |
|------|------|--------|
| locked | totalLevel < node.level | 반투명, 클릭 불가 |
| claimable | totalLevel >= node.level && 미수령 | 골드색 글로우, 클릭 시 수령 |
| claimed | 이미 수령 완료 | 체크 표시(✓) |
| reached | 전환 노드, totalLevel >= node.level | 정상 표시 |

#### 미수령 보상 영역
- 미수령 마일스톤이 있을 때만 표시
- 좌: "미수령 보상: N개" (골드색)
- 우: [모두 수령] 버튼 — 클릭 시 도달한 미수령 마일스톤 일괄 수령

### 1-4. 강화 버튼 (가로 3열)

레퍼런스 이미지 하단의 HP/공격력/방어력 카드형 버튼과 동일한 구조.
**버튼 안에 모든 정보가 포함**되어야 하며, **가로 배치**.

```
┌──────────┐ ┌──────────┐ ┌──────────┐
│    ⚔️     │ │    ❤️     │ │    🛡️     │
│   공격력   │ │   체력    │ │   방어력   │
│ Lv.3 / 10 │ │ Lv.5 / 10│ │ Lv.2 / 10 │
│    +3     │ │   +15    │ │    +2     │
│  240 🪙   │ │  240 🪙  │ │  240 🪙   │
└──────────┘ └──────────┘ └──────────┘
```

각 버튼 내부 (위→아래):
1. 스탯 아이콘 (공격/체력/방어)
2. 스탯 이름 (공격력/체력/방어력) — 볼드
3. 서브 등급 내 레벨 (`Lv.{tierLevel} / {levelsPerStat}`) — MAX 시 초록색
4. 레벨당 증가분 (`+{perLevel}`) — 초록색 볼드
5. 비용 또는 MAX — 골드색

- 서브 등급 내 스탯별 최대 10레벨 (`levelsPerStat`)
- 3개 스탯 모두 10 도달 시 서브 등급 진급 + 레벨 리셋
- 비용은 서브 등급 내 고정 (3개 스탯 동일)
- disabled 조건: 스탯 MAX(10 도달) 또는 골드 부족

### 1-5. 유산 카드 (조건부)
```
┌────────────────────────────────┐
│ 유산                           │
│ 경로          {route}          │
│ 레벨          {level}          │
└────────────────────────────────┘
```
- `Player.isHeritageUnlocked()` = true일 때만 표시 (영웅 등급 도달)

### 1-6. 보상 수령 팝업 (modal-overlay)
```
┌────────────────────────────────┐
│             🪙                 │
│                                │
│         보상 수령!              │
│      골드 +720                 │
│                                │
│     [        확인        ]     │
└────────────────────────────────┘
```
- 개별 마일스톤 클릭 수령 시 표시
- 팝업 외부(overlay) 클릭 시 닫힘
- [확인] 버튼 클릭 시 닫힘

## 2. 상호작용

| 요소 | 액션 | 결과 |
|------|------|------|
| 강화 버튼 (공/체/방) | 클릭 | 해당 스탯 1레벨 강화, 골드 차감, 세이브 |
| 강화 버튼 (disabled) | 클릭 | 반응 없음 (MAX 또는 골드 부족) |
| 마일스톤 아이콘 (claimable) | 클릭 | 보상 수령 + 수령 팝업 표시 |
| 마일스톤 아이콘 (locked/claimed) | 클릭 | 반응 없음 |
| [모두 수령] 버튼 | 클릭 | 미수령 마일스톤 일괄 수령, 세이브 |
| 팝업 overlay | 클릭 | 팝업 닫힘 |
| [확인] 버튼 | 클릭 | 팝업 닫힘 |

### 서브 등급 진급 흐름
1. 공격력/체력/방어력 각각 10레벨 도달
2. 마지막 10번째 강화 시 자동 진급
3. 3개 스탯 레벨 → 0으로 리셋
4. 서브 등급 +1 (예: 모험가 2단 → 모험가 3단)
5. 강화 비용 상승 (다음 서브 등급 고정 비용)

## 3. 데이터 소스

| UI 요소 | 데이터 소스 |
|---------|------------|
| 등급 라벨 | `TalentTable.getSubGradeLabel(totalLevel)` |
| 서브 등급 진행 | `TalentTable.getSubGradeInfo(totalLevel)` |
| 다음 메인 등급 경계 | `talent.getNextGradeThreshold()` |
| 골드 획득량 % | `Player.getGoldMultiplier()` |
| 스탯 레벨 (서브 등급 내) | `talent.getStatLevelInTier(stat)` |
| 레벨당 증가분 | `TalentTable.getStatPerLevel(stat)` |
| 스탯당 최대 레벨 | `TalentTable.getLevelsPerStat()` (10) |
| 강화 비용 | `talent.getUpgradeCost(stat)` |
| 마일스톤 목록 | `TalentTable.getAllMilestones()` |
| 등급 전환 목록 | `TalentTable.getAllTransitions()` |
| 미수령 마일스톤 | `talent.getClaimableMilestones(claimedMilestones)` |
| 슬라이딩 윈도우 범위 | `VIEW_RANGE = 15`, `TalentTable.getMaxLevel()` |
