## 교체 방법

### 아이콘 (§1~3, UI 공통 스프라이트)
1. 동일 파일명으로 PNG 준비
2. `Assets/_Project/Resources/Icons/` 폴더에 덮어쓰기
3. Unity 열기 → 자동 reimport → **즉시 반영** (Inspector 작업 불필요)

SpriteManager가 `Resources.LoadAll<Sprite>("Icons")`로 자동 로드.
파일명(확장자 제외) = 아이콘 ID (예: `icon_atk.png` → ID `"icon_atk"`).

### 캐릭터/장비/펫/상태이상 스프라이트 (§4~7)
1. 이미지 파일을 프로젝트에 추가
2. SpriteDatabase ScriptableObject(`Resources/SpriteDatabase`)에서 Inspector로 할당
   - `characters[]`: id + idle sprite + hit sprite
   - `equipmentIcons[]`: slot + grade + icon sprite
   - `petIcons[]`: petId + tier + icon sprite
   - `statusEffects[]`: type + icon sprite
