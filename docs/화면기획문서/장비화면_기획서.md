```
장비
```

### 1-2. 탭 바
```
[ ⚔무기 ] [ 🛡방어구 ] [ 💍반지 ] [ 📿목걸이 ] [ 👟신발 ] [ 🧤장갑 ] [ 🎩모자 ] [ 합성 (N) ]
```

- 탭 바는 flexWrap으로 줄바꿈 허용
- 기본 탭: 무기

## 2. 탭별 화면 구성

### 2-1. 슬롯 탭 (무기/방어구/반지/목걸이/신발/장갑/모자)

각 슬롯 탭은 동일한 구조로, 해당 슬롯의 장착 아이템과 보관함 아이템을 한 화면에서 표시한다.
장착/보관함 아이템에 슬롯별 SVG 아이콘이 표시된다.

#### 재화 정보 카드
```
┌────────────────────────────────┐
│ 강화석                   {수량} │
└────────────────────────────────┘
```

#### 장착 중 (장착수/최대)
```
┌────────────────────────────────┐  ← 장착 슬롯 (빨간 테두리)
│ [등급색상] 장비명               │
│ {등급명} Lv.N | ATK +N HP +N  │
│                    [강화] [해제] │
└────────────────────────────────┘
```

| 슬롯 | 최대 장착 수 |
|------|-------------|
| 무기 | 1 |
| 방어구 | 1 |
| 반지 | 2 |
| 목걸이 | 1 |
| 신발 | 1 |
| 장갑 | 1 |
| 모자 | 1 |

- S등급 장비: `[S]` 접두사 표시
- 빈 슬롯: "빈 슬롯" 텍스트
- **강화 버튼**: 장비 강화석 1개 소모, 강화석 부족 시 비활성
- **해제 버튼**: 장비를 보관함으로 이동
- 장착 중인 슬롯은 빨간 테두리(`#e94560`)로 강조
- **클릭 시 패시브 능력 표시**: 장착 장비 클릭 시 패시브 능력 설명 패널 확장

#### 보관함 (N)
```
┌────────────────────────────────────┐
│ [등급색상] 장비명                   │
│ {등급명} Lv.N | ATK +N HP +N      │
│                     [장착] [{가격}G] │
└────────────────────────────────────┘
```

- 해당 슬롯의 보관함 아이템만 표시
- 정렬 기준: 등급 높은 순 → S등급 우선 → 레벨 높은 순
- **장착 버튼**: 빈 슬롯 우선 장착, 없으면 가장 약한 장비 교체
- **판매 버튼**: `{가격}G` 표시
- S등급 장비: `판매불가` 비활성 버튼
- 빈 보관함: "보관 중인 {슬롯명}이 없습니다" 표시
- **클릭 시 패시브 능력 표시**: 보관함 장비 클릭 시 패시브 능력 설명 패널 확장

#### 판매 가격표

| 등급 | 판매가 |
|------|--------|
| 일반 | 10G |
| 우수 | 30G |
| 희귀 | 100G |
| 에픽 | 300G |
| 전설 | 1000G |
| 신화 | 3000G |

### 2-2. 합성 탭

#### 그룹 카드 (접힌 상태)
```
┌──────────────────────────────────┐
│ [등급색상] {등급명} {슬롯명}  N/3 │
│ ████████░░░░░░░░ (진행 바)       │
│ {합성 가능! or N개 더 필요} → 결과│
└──────────────────────────────────┘
```

- 보관함 장비를 **등급+슬롯(+무기종류)** 기준으로 그룹핑
- 무기는 검/지팡이/활 종류가 같아야 합성 가능
- 그룹 정렬: 등급 높은 순 → 슬롯명 알파벳 순
- S등급 장비: 그룹에서 제외
- 합성 불가 등급(신화): 그룹에서 제외
- 진행 바: 합성 가능 시 초록, 미달 시 빨강
- 결과 미리보기: 합성 시 생성될 등급 표시 (등급 색상 적용)

#### 그룹 카드 (펼친 상태 - 클릭 시)
```
┌──────────────────────────────────┐
│ [등급색상] {등급명} {슬롯명}  N/3 │
│ ████████░░░░░░░░ (진행 바)       │
│ ──────────────────────────────── │
│ [아이콘]+[아이콘]+[아이콘]=[결과]│
│ (보유=실선, 부족=점선 반투명)    │
│ ──────────────────────────────── │
│ 합성 결과     {결과 등급+슬롯}   │
│ 패시브 변화   {패시브 설명}      │
│                                  │
│ 보유 장비 (N개)                   │
│ ┌ [등급색상] 장비명   Lv.N ─────┐│
│ └ [등급색상] 장비명   Lv.N ─────┘│
│                                  │
│ [     합성하기 / 재료 부족     ] │
└──────────────────────────────────┘
```

- 카드 클릭 시 상세 패널 토글
- **아이콘 다이어그램**: 보유 장비=실선 테두리 불투명, 부족=점선 테두리 반투명
- **합성 결과/패시브 변화**: 결과 등급과 패시브 능력 미리보기
- **보유 장비 리스트**: 해당 그룹의 모든 장비 표시
- **합성하기 버튼**: 재료 충분 시 활성 (primary), 부족 시 "재료 부족" 비활성
- 합성 실행 시: 재료 소모 → 결과 장비 인벤토리 추가 → 자동 저장

## 3. 합성 규칙

| 재료 등급 | 필요 수량 | 결과 등급 |
|-----------|----------|-----------|
| 일반 | 3 | 우수 |
| 우수 | 3 | 희귀 |
| 희귀 | 3 | 에픽 |
| 에픽 | 3 | 전설 |
| 전설 | 3 | 신화 |

- 모든 등급 동일하게 같은 등급 3개 합성 → 다음 등급
- S등급 장비는 합성 재료로 사용 불가

## 4. 데이터 매핑

### 슬롯 탭

```
슬롯 목록    → Player.getEquipmentSlot(slotType)
장착 장비    → EquipmentSlot.equipped[]
보관함 장비  → Player.inventory.filter(e => e.slot === slotType)
장비 스탯    → Equipment.getStats() → { atk, maxHp }
강화석       → Player.resources.equipmentStones
장착 처리    → Player.equipFromInventory(id)
해제 처리    → Player.unequipToInventory(slotType, index)
판매 처리    → Player.sellEquipment(id)
판매 가격    → SELL_PRICES[grade]
```

### 합성 탭

```
합성 후보    → Forge.findMergeCandidates(inventory)
합성 가능여부 → Forge.canMerge(equipments)
합성 실행    → Forge.merge(equipments)
필요 수량    → EquipmentTable.getMergeCount(grade)
다음 등급    → EquipmentTable.getNextGrade(grade)
등급 인덱스  → EquipmentTable.getGradeIndex(grade)
```

## 6. 상호작용 흐름

### 장비 강화
```
슬롯 탭 → [강화] 클릭 → 강화석 1개 소모 → 장비 레벨 +1 → 슬롯 레벨 동기화 → 자동 저장
```
- 강화 레벨은 슬롯에 귀속 (장비 교체/해제해도 슬롯 레벨 유지)
- 새 장비 장착 시 슬롯 레벨이 자동 적용됨

### 장비 해제/장착
```
슬롯 탭 장착 영역 → [해제] → 현재 레벨 유지한 채 보관함으로 이동 → 자동 저장
슬롯 탭 보관함 영역 → [장착] → 빈 슬롯 또는 최약 장비 교체 → 슬롯 레벨 적용 → 자동 저장
```

### 장비 판매
```
슬롯 탭 보관함 영역 → [{가격}G] → 골드 획득 → 주간퀘스트(weekly_sell) 진행 → 자동 저장
```

### 장비 합성
```
합성 탭 → 그룹 카드 클릭 → 상세 패널 확인 → [합성하기] 클릭
→ 재료 장비 인벤토리에서 제거 → 결과 장비 인벤토리에 추가 → 자동 저장
```
