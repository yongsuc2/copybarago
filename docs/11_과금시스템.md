# 카피바라고! 역기획서 - 과금/수익화 시스템

## 수익화 모델

카피바라고는 **F2P + 광고 + IAP(인앱결제)** 하이브리드 모델을 채택한다.

## 과금 상품 구조

### 1. 월정액 (Monthly Pass)

| 항목 | 내용 |
|------|------|
| 핵심 혜택 | 탐방(Travel) 50배속 해금 |
| 추가 혜택 | 일일 보석/스태미나 지급 (추정) |
| 가치 | 골드 파밍 효율 극대화 |
| 대상 유저 | 소과금 ~ 중과금 |

### 2. 보석 직접 구매

| 항목 | 내용 |
|------|------|
| 용도 | 보물상자 소환, 경기장 재도전 등 |
| 번들 | 소량 ~ 대량 번들 (가격에 따라 보너스 보석) |
| 대상 유저 | 중과금 ~ 고과금 |

### 3. 보물상자 키/패키지

| 항목 | 내용 |
|------|------|
| 용도 | 가챠 직접 진행 |
| 구성 | 키 단품 또는 재화 번들 패키지 |
| 대상 유저 | 중과금 ~ 고과금 |

### 4. 이벤트 한정 패키지

| 항목 | 내용 |
|------|------|
| 구성 | 한정 장비/펫 + 보석 + 재화 번들 |
| 시기 | 명절/기념일 이벤트 기간 |
| 대상 유저 | 중과금 ~ 고과금 |

## 광고 수익 모델

### 광고 시청 보상

| 광고 유형 | 보상 | 일일 제한 |
|-----------|------|-----------|
| 스태미나 충전 | 스태미나 N개 | 제한 있음 |
| 도전권 충전 | 도전권 1장 | 제한 있음 |
| 보물상자 오픈 | 무료 상자 오픈 | 제한 있음 |
| 던전 티켓 충전 | 던전 재입장 | 제한 있음 |
| 보상 2배 | 클리어 보상 2배 | 제한 있음 |

### 무과금 유저의 광고 의존도
```
광고 시청 = F2P 유저의 핵심 자원 획득 수단
광고 미시청 시 성장 속도 대폭 감소
→ 광고 시청 = 사실상 필수
```

## 과금 효율 분석 (추정)

### 과금 효율 순위
```
1순위: 월정액 - 가성비 최고 (탐방 50배속 = 골드 파밍 효율 극대화)
2순위: 한정 이벤트 패키지 - 일반 구매 대비 높은 가치
3순위: 보석 대량 번들 - 한정 상자 천장용
4순위: 보석 소량 번들 - 비효율
```

### 과금 벽 (Pay Wall) 분석
```
콘텐츠 자체: 과금 벽 낮음 (모든 콘텐츠 무과금 접근 가능)
성장 속도: 과금 벽 높음 (탐방 배율, 보석 획득량 차이)
PvP 경쟁력: 과금 벽 중간 (장비/특성 차이)
```

## AI Agent 과금 관련 로직

### 무과금 모드 (기본)
```
ALWAYS: 모든 광고 시청 기회 활용
ALWAYS: 무료 보상 전부 수령
NEVER: 보석으로 스태미나/도전권 구매 (비효율)
FOCUS: 보석은 한정 상자 천장 전용으로 비축
```

### 소과금 모드
```
IF 월정액_구매 THEN
  탐방 50배속으로 골드 파밍 극대화
  일일 보석 지급으로 천장 도달 주기 단축
```

### 중과금 이상 모드
```
한정 이벤트 패키지 선별 구매
보석 대량 번들로 한정 상자 집중 공략
S-Epic 장비 조기 확보 → 성장 가속
```

## 수익화 시스템 요약도

```
[무과금 유저]
    └→ 광고 시청 → 스태미나/도전권/상자 → 느린 성장

[소과금 유저]
    └→ 월정액 → 탐방 50배속 → 골드 파밍 가속 → 중간 성장

[중과금 유저]
    └→ 보석 구매 → 한정 상자 천장 → S-Epic 조기 확보 → 빠른 성장

[고과금 유저]
    └→ 전 패키지 구매 → 최대 효율 성장 → PvP 상위 랭크
```
