# 카피바라고! 역기획서 - 가챠(보물상자) 시스템

## 개요

카피바라고의 가챠는 **보물상자(Treasure Chest)** 시스템으로 운영된다.
영웅 소환이 아닌 **장비 및 소장품** 획득이 주 목적이다.

## 보물상자 종류

| 상자 | 주요 보상 | 등급 |
|------|-----------|------|
| 브론즈 상자 | 일반~우수 장비, 기본 재화 | 하위 |
| 실버 상자 | 우수~희귀 장비, 재화 | 중하위 |
| 골드 상자 | 희귀~에픽 장비, 소장품 | 중상위 |
| 펫 상자 | 펫 알, 펫 사료 | 특수 |
| 보석 상자 | 보석 중심 보상 | 특수 |

## 소환 비용

| 소환 유형 | 비용 |
|-----------|------|
| 단일 소환 (1회) | 보석 298개 (추정) |
| 10연차 (10회) | 보석 2,980개 |
| 10연차 할인 | 9회 가격에 10회 (1회 무료) |

## Limited Treasure Chest (한정 상자)

| 항목 | 내용 |
|------|------|
| 천장 시스템 | 180회 뽑기 |
| 천장 보상 | S-Epic 장비 확정 획득 |
| 전략 | 한정 상자에 보석 집중 투자 |

```
[소환 카운트]
0회 ────────────────────── 180회
│                           │
│  일반 확률 드롭             │
│  에픽/전설 확률 드롭         │
│                           │
└──────────────────────── S-Epic 확정
```

## 보상 구조

### 일반 보물상자 보상 풀

```
보물상자 오픈
  ├ 장비 (확률 분포)
  │   ├ 일반 ~ 우수: 높은 확률
  │   ├ 희귀: 중간 확률
  │   ├ 에픽: 낮은 확률
  │   └ S-에픽: 매우 낮은 확률
  ├ 소장품
  │   └ 천외 운철 포함
  └ 재화
      ├ 골드
      ├ 장비 강화석
      └ 기타 소모품
```

### 포인트 누적 시스템
- 소환 시 포인트 누적
- 일정 포인트 도달 시 추가 상자 획득
- 10연차 시 포인트 효율 극대화

## 소장품 시스템

| 항목 | 내용 |
|------|------|
| 획득 경로 | 보물상자, 이벤트 |
| 효과 | 도감 등록 시 영구 스탯 보너스 |
| 전략 | 중복 소장품도 보너스 누적 가능 여부 확인 필요 |

## AI Agent 가챠 전략

### 보석 사용 우선순위
```
1순위: Limited Treasure Chest (한정 상자) - S-Epic 천장 노림
2순위: 이벤트 한정 상자 - 한정 보상
3순위: 상시 골드 상자 - 장비 파밍
최하위: 브론즈/실버 상자 - 비효율
```

### 천장 관리 로직
```
IF 한정_상자_카운트 > 0 AND 한정_상자_카운트 < 180 THEN
  현재 한정 상자에 계속 투자 (천장까지)

IF 보석 < 한정_상자_천장까지_필요량 THEN
  보석 비축 (다음 한정 상자 대비)

NEVER: 천장 중간에 다른 상자로 전환
```

### 10연차 규칙
```
ALWAYS: 10연차 단위로 소환 (1회 무료 혜택)
NEVER: 단일 소환 (비효율)
```

### 무과금 보석 관리
```
일일_보석_수입 추정:
  - 거대 드래곤 둥지: X 보석
  - 일일 퀘스트: Y 보석
  - 이벤트: Z 보석

한정_상자_주기에 맞춰 보석 비축 목표 설정
180회 × 298 = 약 53,640 보석 (천장 도달 비용)
```
