# 카피바라고! 역기획서 - 가챠(보물상자) 시스템

## 개요

카피바라고의 가챠는 **장비 뽑기** 시스템으로 운영된다.
합성 비율(3:1)에 기반한 정직한 확률 분배를 사용한다.

## 보물상자 종류

| 상자 | 주요 보상 | 비용 |
|------|-----------|------|
| 장비 상자 | 일반~신화 장비 | 150 보석 |
| 펫 상자 | 펫 알, 펫 사료 | 298 보석 |
| 보석 상자 | 보석 20~100 | 특수 |

## 장비 상자 확률

합성 비율 3:1 기반 — 각 등급은 이전 등급 대비 3배 희귀하다.

| 등급 | 가중치 | 확률 | 합성 환산 (일반 기준) |
|------|--------|------|---------------------|
| 일반 | 243 | 66.76% | 1 |
| 우수 | 81 | 22.25% | 3 |
| 희귀 | 27 | 7.42% | 9 |
| 에픽 | 9 | 2.47% | 27 |
| 전설 | 3 | 0.82% | 81 |
| 신화 | 1 | 0.27% | 243 |

기대값: 뽑기 1회 = 일반 장비 4개 가치 (모든 등급이 동일한 기대값 기여)

## 소환 비용

| 소환 유형 | 비용 |
|-----------|------|
| 단일 소환 (1회) | 보석 150개 |
| 10연차 (10회) | 보석 1,350개 (1회 무료) |

## 신화 확정 시스템

| 항목 | 내용 |
|------|------|
| 신화 확정 횟수 | 180회 |
| 신화 확정 보상 | 신화 장비 확정 |
| 신화 확정 비용 | 180 × 150 = 27,000 보석 |

```
[소환 카운트]
0회 ────────────────────── 180회
│                           │
│  합성 비율 기반 확률 드롭    │
│                           │
└──────────────────────── 신화 확정
```

## S-등급 시스템

| 항목 | 내용 |
|------|------|
| 대상 등급 | 에픽 이상 (에픽/전설/신화) |
| 확률 | 2% |
| 특성 | 판매 불가, 합성 재료 불가 |

## 펫 상자

| 항목 | 내용 |
|------|------|
| 비용 | 298 보석 |
| 보상 | 펫 알 1개 + 펫 사료 1~4개 |
| 천장 | 없음 |
