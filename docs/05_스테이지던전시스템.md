# 카피바라고! 역기획서 - 스테이지/던전 시스템

## 메인 콘텐츠: 챕터

### 챕터 진입 조건
- 스태미나 5개 소모

### 챕터 유형별 구조

#### 60일 챕터
```
[진입] → [인카운터 60턴 반복] → [보스 전투] → [클리어]
                ↓
        [중박/대박 룰렛]
        - 중박: 전설 등급 스킬 획득
        - 대박: 불멸 등급 스킬 획득 (천사의 힘, 악마의 힘)
```

#### 30일 챕터
```
[진입] → [인카운터 30턴 반복] → [보스 전투] → [클리어]
                ↓
        [행운 상인] → 행운 코인 5개 지급
                ↓
        [행운 머신 선택]
        ├ 100% 성공률 머신 (낮은 보상)
        ├ 70% 성공률 머신 (중간 보상)
        └ 40% 성공률 머신 (높은 보상)
```

#### 5일 챕터
```
[진입] → [웨이브 전투 5턴] → [클리어]
        - 누적 턴 시스템
        - 초보자 대상 콘텐츠
```

## 도전 콘텐츠

### 도전의 타워 (Tower)

| 항목 | 내용 |
|------|------|
| 규모 | 100층, 각 층 10단계 |
| 진입 비용 | 도전권 1장 |
| 실패 시 | 도전권 미소모, 재시도 가능 |
| 보상 주기 | 5단계, 10단계마다 |
| 핵심 보상 | 도면(Power Stone) - 장비 승급 필수 재화 |

```
[층 진입] → [1~4단계: 일반 전투] → [5단계: 중간 보상(도면)]
    → [6~9단계: 강화 전투] → [10단계: 보스 + 보상(도면)]
    → [다음 층]
```

### 지하감옥 돌진

| 항목 | 내용 |
|------|------|
| 시작점 | 과거 최고 기록 층에서 시작 |
| 구성 | 각 층 5전투 + 보스 전투 |
| 특징 | 연속 도전, 중단 없는 진행 |
| 보상 | 층수에 비례한 보상 |

```
[자동 시작 (최고 기록)] → [5전투 연속] → [보스] → [다음 층] → [반복]
                                                         ↓
                                                   [패배 시 종료]
                                                   [최고 기록 갱신]
```

### 고블린 광부

| 항목 | 내용 |
|------|------|
| 도구 | 곡괭이 (일일 지급) |
| 행동 | 벽돌을 부수는 채굴 미니게임 |
| 목표 | 광석 30개 수집 |
| 보상 | 행운의 광차 이용권 |

```
[곡괭이 소모] → [벽돌 채굴] → [광석 획득]
                                ↓
                    [30개 달성] → [행운의 광차] → [보상]
```

|------|------|

## AI Agent 일일 루틴 최적화

### 우선순위 스케줄

```
[일일 루틴 시작]
    ↓
1순위: 일일 던전 전부 소화
    ├ 거대 드래곤의 둥지 (스태미나/보석)
    ├ 아스트랄 나무 (펫 재화)
    └ 환검의 하늘섬 (보물상자)
    ↓
2순위: 도전의 타워 (도전권 소모)
    ↓
3순위: 지하감옥 돌진 (최고 기록 갱신 시도)
    ↓
4순위: PvP 경기장 (입장권 소모)
    ↓
5순위: 메인 챕터 진행 (스태미나 소모)
    ↓
6순위: 탐방(Travel) (잔여 스태미나 전부 소모)
    ↓
7순위: 고블린 광부 (곡괭이 소모)
    ↓
[일일 루틴 종료]
```

### 타워 도전 판단
```
IF 현재_전투력 > 해당_층_권장_전투력 THEN 도전
IF 5연속_실패 THEN 중단 (전투력 부족 판정)
```

### 챕터 선택 로직
```
IF 새_챕터_해금됨 THEN 새 챕터 도전
IF 클리어_실패 2회 THEN 탐방으로 전환 (자원 파밍)
```
