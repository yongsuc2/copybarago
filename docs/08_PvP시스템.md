# 카피바라고! 역기획서 - PvP 시스템 (경기장)

## 개요

| 항목 | 내용 |
|------|------|
| 명칭 | 경기장 (Arena) |
| 유형 | 비동기 PvP (실시간 대전 아님) |
| 매칭 | 같은 티어 유저 4명 매칭 |
| 진입 비용 | 입장권 |
| 추가 도전 | 보석으로 재도전 가능 |

## 티어 체계

```
브론즈 → 실버 → 골드 → 플래티넘 → 다이아 → 마스터
```

| 티어 | 승급 조건 | 보상 수준 |
|------|-----------|-----------|
| 브론즈 | 초기 배치 | 기본 |
| 실버 | 브론즈 상위 랭크 | 낮음 |
| 골드 | 실버 상위 랭크 | 중간 |
| 플래티넘 | 골드 상위 랭크 | 중상 |
| 다이아 | 플래티넘 상위 랭크 | 높음 |
| 마스터 | 다이아 상위 랭크 | 최상 |

## 전투 구조

```
[입장권 소모] → [같은 티어 유저 4명 매칭]
    ↓
[자동 전투] → 플레이어 빌드 기반 자동 교전
    ↓
[순위 결정] → 4명 중 순위 산정
    ↓
[보상 지급] → 순위에 따른 보상
    ↓
[티어 포인트 변동] → 승급/강등 반영
```

## 전투 메커니즘

| 항목 | 내용 |
|------|------|
| 전투 방식 | 턴제 자동 전투 (PvE와 동일) |
| 적용 스탯 | 특성 + 장비 + 전승 (영구 성장만 적용) |
| 스킬 | 챕터 스킬 미적용, 장비/전승 효과만 |
| 승리 기준 | 상대 HP 0 |

## AI Agent PvP 전략

### 도전 판단
```
ALWAYS: 일일 입장권 전부 소모 (무조건 도전)
IF 보석_여유 AND 승률 > 60% THEN 보석으로 추가 도전
IF 승률 < 40% THEN 추가 도전 자제 (보석 보존)
```

### 전투력 기반 전략
```
PvP에서는 챕터 스킬이 적용되지 않으므로:
- 영구 성장(특성/장비/전승) 투자가 PvP 직결
- 장비 등급 우선 강화
- 특성 ATK > HP 우선
```

### 티어 관리
```
티어_승급 시 상대 강해짐 → 무리한 승급보다 안정적 보상 수급
IF 현재_티어에서 승률 > 70% THEN 승급 시도
IF 승급_직후 승률 < 30% THEN 전투력 강화 우선
```
