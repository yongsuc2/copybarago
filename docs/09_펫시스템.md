# 카피바라고! 역기획서 - 펫 시스템

## 개요

| 항목 | 내용 |
|------|------|
| 역할 | 전투 보조 + 패시브 효과 |
| 장착 | 활성 펫 슬롯에 장착하여 전투 참여 |
| 특징 | 미사용 펫도 업그레이드 시 패시브 보너스 제공 |
| 육성 재화 | 펫 알, 사료 |
| 주요 획득처 | 아스트랄 나무 (Celestial Tree) 던전 |

## 펫 등급 체계

```
일반 → 희귀 → 에픽 → 전설(Legendary) → 불멸(Immortal)
```

## 펫 육성 구조

### 레벨업
```
[펫 사료 투입] → [펫 경험치 상승] → [펫 레벨 상승] → [스탯 증가]
```

### 등급 승급
```
[동일 펫 중복 획득] → [합성/승급] → [다음 등급]
일반 → 희귀 → 에픽 → 전설 → 불멸
```

### 패시브 보너스 (미사용 펫 포함)
```
모든 보유 펫의 업그레이드 레벨에 따라
글로벌 패시브 보너스 적용
→ 사용하지 않는 펫도 반드시 업그레이드
```

## 펫 획득 경로

| 경로 | 획득 대상 |
|------|-----------|
| 아스트랄 나무 (일일 던전) | 펫 알, 사료 |
| 펫 상자 (가챠) | 펫 알 |
| 이벤트 보상 | 특정 펫/사료 |

## 펫 알 부화

```
[펫 알 획득] → [부화] → [랜덤 펫 획득]
                          ├ 일반 펫: 높은 확률
                          ├ 희귀 펫: 중간 확률
                          ├ 에픽 펫: 낮은 확률
                          └ 전설/불멸 펫: 매우 낮은 확률
```

## AI Agent 펫 관리 전략

### 활성 펫 선택
```
1순위: 보유 중 가장 높은 등급의 S티어 펫
2순위: S티어 없으면 A티어 최고 등급
3순위: 빌드 시너지 고려
  - 해골/유협 계승 → 공격형 펫 (Freya, Flash)
  - 기사 계승 → 방어형 펫 (Slime King, Unicorn)
  - 유령 계승 → 속성 펫 (Elsa, Ice Wind Fox)
```

### 사료 배분
```
ALWAYS: 활성 펫 최우선 사료 투입
THEN: 비활성 펫도 순차적 업그레이드 (패시브 보너스)
```

### 펫 알 관리
```
ALWAYS: 즉시 부화 (보관 이점 없음)
중복 펫: 즉시 합성/승급에 사용
```

### 아스트랄 나무 우선순위
```
일일 던전 중 아스트랄 나무 = 최우선 소화 대상
펫 육성 재화의 유일한 안정적 공급원
```
