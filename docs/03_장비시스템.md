# 카피바라고! 역기획서 - 장비 시스템

## 장비 슬롯 구성

| 슬롯 | 착용 수 | 주요 스탯 |
|------|---------|-----------|
| 무기 | 1개 | ATK |
| 방어구 | 1개 | HP |
| 반지 | 2개 | ATK |
| 목걸이 | 1개 | ATK+HP |
| 신발 | 1개 | HP |
| 장갑 | 1개 | ATK |
| 모자 | 1개 | HP |

총 8개 슬롯 운용 (반지 2개 포함)

## 무기 종류

무기 슬롯은 3종류로 세분화된다. 장비 생성 시 랜덤으로 결정되며, 같은 종류끼리만 합성 가능하다.

| 종류 | 패시브 효과 | 테마 |
|------|-----------|------|
| 검 | 공격력 % 증가 (BUFF) | 공격력 특화 |
| 지팡이 | 마법 스킬 계수 증가 (MAGIC_BOOST) | 마법 특화 |
| 활 | 분노 추가 충전 (RAGE_BOOST) | 속도/분노 |

## 장비 패시브 능력

모든 장비는 등급에 따라 패시브 능력을 보유한다. 등급이 올라갈수록 패시브 수치가 강해진다. 구체적 수치는 `EquipmentPassiveTable` 데이터 테이블 참조.

| 슬롯 | 패시브 효과 |
|------|-----------|
| 무기(검) | 공격력 % 증가 |
| 무기(지팡이) | 마법 스킬 계수 증가 |
| 무기(활) | 분노 추가 충전 |
| 방어구 | 최대체력% 방어막 |
| 반지 | 치명타 확률 증가 |
| 목걸이 | 공격력 % 증가 |
| 신발 | 매턴 HP 회복 |
| 장갑 | 추가 타격 확률 |
| 모자 | 방어력 % 증가 |

장비 패시브는 전투 시작 시 자동으로 Skill 객체로 변환되어 BattleUnit에 적용된다.

## 장비 등급 체계

```
일반 → 우수 → 희귀 → 에픽 → 전설 → 신화
```

## 장비 강화 시스템

### 레벨 강화

| 항목 | 내용 |
|------|------|
| 재화 | 장비 강화석 |
| 효과 | 장비 기본 스탯 증가 |
| 특징 | 강화 레벨은 슬롯에 귀속 (장비 교체 시 유지) |

### 승급 시스템

| 단계 | 필요 재화 |
|------|-----------|
| 10단계 | 도면(Power Stone) |
| 20단계 | 도면(Power Stone) |
| 30단계 | 도면(Power Stone) |

도면은 도전의 타워 5/10단계 클리어 시 획득

### 슬롯 레벨 유지 규칙
```
무기 슬롯 Lv.25 상태에서 장비 교체 시
→ 새 장비가 자동으로 Lv.25 적용
→ 교체/해제된 장비는 현재 레벨을 유지한 채 보관함으로 이동
→ 보관함 장비를 다시 장착하면 슬롯 레벨로 덮어씌워짐
→ 강화 레벨은 슬롯에 귀속되므로 강화석 손실 없음
```

## 장비 합성 시스템

### 합성 규칙

```
동일 등급 장비 3개 → 다음 등급 장비 1개

일반 × 3 → 우수
우수 × 3 → 희귀
희귀 × 3 → 에픽
에픽 × 3 → 전설
전설 × 3 → 신화
```

- 모든 등급 동일하게 3개 합성
- S등급 장비는 합성 재료로 사용 불가
- 무기는 같은 종류(검/지팡이/활)끼리만 합성 가능

## 장비 인벤토리

미장착 장비는 Player.inventory에 보관된다.

### 장비 판매

| 등급 | 판매가 (골드) |
|------|-------------|
| 일반 | 10 |
| 우수 | 30 |
| 희귀 | 100 |
| 에픽 | 300 |
| 전설 | 1000 |
| 신화 | 3000 |

S등급 장비는 판매 불가 (실수 방지).

### 장착/해제 흐름
- 보관함에서 "장착" → 빈 슬롯 우선, 없으면 가장 약한 장비 교체 (교체된 장비는 보관함으로)
- 장착 탭에서 "해제" → 해당 장비가 보관함으로 이동

## 장비 획득 경로

| 경로 | 획득 장비 등급 |
|------|---------------|
| 챕터 클리어 보상 | 일반 ~ 희귀 |
| 보물상자(가챠) | 우수 ~ S-에픽 |
| 환검의 하늘섬(던전) | 보물상자 (장비 포함) |
| 이벤트 보상 | 에픽 ~ S-에픽 (한정) |
| 합성 | 하위 등급 → 상위 등급 |

## AI Agent 장비 관리 전략

### 장비 장착 우선순위
```
1. S등급 장비 > 일반 장비 (같은 등급 내)
2. 높은 등급 > 낮은 등급
3. 빌드 시너지 고유 효과 > 단순 스탯 우위
```

### 합성 전략
```
1. S등급 장비를 합성 재료로 사용 금지
2. 비-S 장비를 재료로 활용하여 S등급 승급
3. 에픽 이하 비-S 장비는 합성 재료로 즉시 활용
4. 동일 슬롯 장비 우선 합성 (장비 교체 최적화)
```

### 강화석 배분
```
1. 메인 무기(ATK) 최우선 강화
2. 반지(ATK) 2순위
3. 장갑(ATK) 3순위
4. 방어구(HP) 4순위
5. 목걸이(ATK+HP) 5순위
6. 신발(HP) 6순위
7. 모자(HP) 7순위
```

### 도면 사용 판단
```
IF 장비가 S등급 AND 장기 사용 예정 THEN 도면 사용
IF 장비가 비-S AND 곧 교체 예정 THEN 도면 보류
```
