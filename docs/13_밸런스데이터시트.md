# 카피바라고! 밸런스 데이터 시트

> 모든 수치는 데이터 테이블(`src/domain/data/json/`)에서 참조.
> 이 문서는 기획 검토용 요약이며, 실제 적용값은 데이터 테이블이 정본.

## 1. 전투 메카닉

> `battle.data.json`

### 데미지 공식

| 항목 | 값 | 설명 |
|------|-----|------|
| 방어 감소율 | 50% | `ATK - DEF × 0.5` |
| 마법 방어 감소율 | 30% | `ATK - DEF × 0.3` |
| 기본 마법 계수 | 0.5 | 지팡이 MAGIC_BOOST로 가산 |
| 데미지 분산 | 0.9 ~ 1.1 | ±10% 랜덤 |
| 치명타 배율 | 1.5× | |
| 최대 턴 수 | 100 | 초과 시 패배 |

| 공격 당 축적 (적) | 25 |
| 분노 공격 배율 | 1.2× |

|--------|---------|
| 공격 시 발동 | 30% |
