# 카피바라고! 역기획서 - 스킬 시스템

## 개요

스킬은 **챕터 진행 중 획득**하는 일시적 능력이다.
챕터 종료 시 스킬은 초기화되며, 매 챕터마다 새로 조합한다.
전승(Heritage) 시스템이 스킬 효과를 영구적으로 증폭시키는 구조.

스킬 수치 데이터: `src/domain/data/SkillTable.ts`, 전투 계수: `src/domain/data/BattleDataTable.ts` 참조

## 스킬 획득 경로

| 경로 | 설명 |
|------|------|
| 천사 인카운터 (중박) | 2개 중 1개 선택 (일반/전설) |
| 악마 인카운터 (대박) | 최대 HP 20% 대가로 랜덤 스킬 획득 |
| 중박 룰렛 | 중박 12회 누적 시 발동, 스킬/회복/골드 선택 |
| 대박 룰렛 | 대박 7회 누적 시 발동, 신화/불멸 스킬 획득 기회 |

## 스킬 등급 체계

| 등급 | 희귀도 | 획득 난이도 |
|------|--------|------------|
| 일반 | 가장 흔함 | 낮음 |
| 전설 | 희귀 | 중간 |
| 신화 | 매우 희귀 | 높음 |
| 불멸 | 최상위 | 대박 룰렛 전용 |

## AI Agent 스킬 선택 알고리즘

### 선택 우선순위 (최상위 → 최하위)

```
Level 1 (즉시 선택):
  - Revive (미보유 시)
  - 불멸 등급 스킬

Level 2 (높은 우선순위):
  - 현재 계승 루트의 핵심 스킬
  - S티어 Proficiency 스킬 (해당 계승)

Level 3 (중간 우선순위):
  - 현재 계승 루트의 보조 스킬
  - 범용 생존 스킬 (흡혈)
  - 범용 공격 스킬

Level 4 (낮은 우선순위):
  - 비시너지 공격 스킬
  - 일반 버프 스킬

Level 5 (비선택):
  - 현재 계승과 역시너지 스킬
```

### 악마 인카운터 판단 로직
```
IF 현재HP > 최대HP * 0.7 THEN
  IF 핵심_스킬_부족 THEN 악마 선택 (리스크 감수)
  ELSE 스킵
IF 현재HP < 최대HP * 0.5 THEN 스킵
```
