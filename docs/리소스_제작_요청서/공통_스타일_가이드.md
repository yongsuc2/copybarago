# 공통 스타일 가이드

모든 캐릭터/몬스터 리소스에 적용되는 아트 스타일 규칙.

## 핵심 키워드

**덩어리감 / 매트 캐주얼 토이 / 평면 색면 / 굵은 외곽선 / 디테일 최소화**

## 조형 원칙

### 실루엣

- 몸통은 콩/감자 형태의 큰 덩어리 1~2개로 구성
- 실루엣만으로 캐릭터가 읽혀야 함

### 비율 (2등신 SD)

| 부위 | 비율 |
|------|------|
| 머리 | 55~75% (전체 인상 지배) |
| 몸 | 25~45% |
| 목 | 거의 없음 (머리 → 바로 몸) |

### 팔/다리

- 팔: 소시지(원통) 1개 + 손(둥근 물방울) 1개, 짧고 통통
- 다리: 발바닥 포함 두꺼운 짧은 기둥 2개
- 손: 장갑 낀 앞발(미튼), 손가락 표현 없음

## 쉐이딩 — 2톤 룰

| 단계 | 설명 |
|------|------|
| 기본색 | 밝은 면 |
| 그림자 | 한 단계 어두운 면, 큰 덩어리로 툭 얹기 |
| 하이라이트 | (선택) 점/작은 면 1개 |

- 그라데이션 거의 사용하지 않음
- 재질감: 매트(무광) + 말랑한 듯 단단한 캐주얼 토이

## 디테일 최소화

- 줄무늬/털 무늬: 얼굴 1포인트 + 몸 0~1포인트만
- 하이라이트: 점/작은 면 1개만 (플라스틱 스펙큘러 금지)
- 털 결/재질 텍스처 거의 없음
- 얼굴: 눈/코/입이 작고 단순 (점, 짧은 선, 작은 U자)

## 외곽선

- 굵고 균일한 외곽선 → "장난감 같은 덩어리" 인상
- 각진 하이라이트/강한 광택 배제
- 굵은 외곽선 + 평면 색면 중심

## 스프라이트 시트 규격

- 캔버스 크기: 1024 x 1536 px
- 그리드: 4열 x 3행 (프레임당 256 x 512 px)
- Row 0 (상단 4칸): 대기(idle) 모션 4프레임 (미세한 숨쉬기/흔들림)
- Row 1 (중단 4칸): 공격(attack) 모션 4프레임
- Row 2 (하단 4칸): 비워두기 (투명)
- 각 프레임 안에서 캐릭터는 중앙 하단 정렬 (발이 프레임 하단에 닿는 위치)
- 투명 배경 (PNG)
- 45도 측면 뷰 (오른쪽 향함)

## 이미지 생성 공통 프롬프트 (영문)

```
in the same art style as the reference orange cat character.
Thick black outlines, flat cel-shading, transparent background, 45-degree side view facing right.

Grid: 4 columns x 3 rows (each cell 256x512px).
- Row 0 (top): 4 idle animation frames (subtle breathing/swaying)
- Row 1 (middle): 4 [attack] animation frames
- Row 2 (bottom): empty/transparent

Character should be centered at the bottom of each cell.
```

## 네거티브 프롬프트

```
realistic fur texture, heavy gradient shading, strong specular highlights (plastic gloss), complex costume details, thin outlines, realistic proportions
```
