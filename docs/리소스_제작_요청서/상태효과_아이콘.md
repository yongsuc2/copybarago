# 상태효과 아이콘 리소스 요청서

## 개요

전투 중 캐릭터에 적용되는 상태효과(버프/디버프)를 표시하는 아이콘.
캐릭터 체력바 아래에 가로로 나열되며, 아이콘 위에 남은 턴 수가 표시된다.

## 저장 경로

```
Assets/_Project/Resources/StatusEffects/
├── status_poison.png
├── status_burn.png
├── status_regen.png
├── status_atk_up.png
├── status_atk_down.png
├── status_def_up.png
├── status_def_down.png
├── status_crit_up.png
└── status_stun.png
```

## 아이콘 목록

| 파일명 | 상태효과 | 종류 | 설명 | 존재 |
|--------|---------|------|------|------|
| `status_poison.png` | 독 (POISON) | 디버프 | 매 턴 지속 피해 | ✅ |
| `status_burn.png` | 화상 (BURN) | 디버프 | 매 턴 지속 피해 | ✅ |
| `status_regen.png` | 재생 (REGEN) | 버프 | 매 턴 체력 회복 | ✅ |
| `status_atk_up.png` | 공격력 증가 (ATK_UP) | 버프 | 공격력 상승 | ✅ |
| `status_atk_down.png` | 공격력 감소 (ATK_DOWN) | 디버프 | 공격력 하락 | ✅ |
| `status_def_up.png` | 방어력 증가 (DEF_UP) | 버프 | 방어력 상승 | ✅ |
| `status_def_down.png` | 방어력 감소 (DEF_DOWN) | 디버프 | 방어력 하락 | ✅ |
| `status_crit_up.png` | 치명타 증가 (CRIT_UP) | 버프 | 치명타 확률 상승 | ✅ |
| `status_stun.png` | 기절 (STUN) | 디버프 | 행동 불가 | ✅ |

## 규격

- 해상도: 128 x 128 px (정사각형)
- 포맷: PNG (투명 배경)
- 스타일: 공통 스타일 가이드 준수 (굵은 외곽선, 평면 색면, 매트 캐주얼)

## 색상 가이드

| 종류 | 배경/테두리 색감 |
|------|----------------|
| 버프 | 초록/파랑 계열 (긍정적 효과) |
| 디버프 | 빨강/보라 계열 (부정적 효과) |

## 아이콘 디자인 제안

| 상태효과 | 아이콘 모티브 |
|---------|-------------|
| 독 | 보라색 해골/독방울 |
| 화상 | 주황색 불꽃 |
| 재생 | 초록색 하트/십자 |
| 공격력 증가 | 빨간 위쪽 화살표/검 |
| 공격력 감소 | 파란 아래쪽 화살표/부러진 검 |
| 방어력 증가 | 파란 방패/위쪽 화살표 |
| 방어력 감소 | 빨간 깨진 방패/아래쪽 화살표 |
| 치명타 증가 | 노란 별/번개 |
| 기절 | 노란 별(어지러움)/소용돌이 |
