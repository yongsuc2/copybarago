## 프롬프트

### 기본 (스프라이트 시트)

```
Create a 1024x1536 pixel sprite sheet of a cute chibi orange cat character. The cat should have a chunky, blob-like body shape (2-head tall SD proportions), big round head, small dot eyes, simple mouth, short stubby arms and legs (both arms visible), wearing a brown short cape/hood with a star brooch on the chest center. Thick black outlines, flat cel-shading with 2-tone shading only, matte casual toy feel, transparent background, 45-degree side view facing right.

Grid: 4 columns x 3 rows (each cell 256x512px).
- Row 0 (top): 4 walking animation frames (cute waddling walk cycle)
- Row 1 (middle): 4 attack animation frames (swinging weapon forward)
- Row 2 (bottom): empty/transparent

Character should be centered at the bottom of each cell.
```

### 기본 포즈 (단일 이미지, 무기 없음)

```
2D game character sprite concept, cute cat, chunky blob-like simple shape, thick clean outlines, short stubby arms and legs (both arms visible), small dot eyes and simple mouth, no fur detail, brown short cape/hood (neutral fantasy), star brooch on chest center, 2-3 tone shading, flat colors, front-facing 3/4 view, transparent background, high resolution
```

### 검 버전

```
(기본 포즈) + right paw holding a short sword, cylindrical sword handle, simple star/circle guard, left arm slightly raised to the side (arm presence emphasis)
```
