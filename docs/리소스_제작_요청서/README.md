## 사용법

1. 공통_스타일_가이드.md를 먼저 읽고 스타일 숙지
2. 해당 리소스 문서의 프롬프트를 ChatGPT에 복사
3. 레퍼런스 이미지(기존 player.png)를 함께 첨부
4. 생성된 파일을 `Assets/_Project/Resources/Chars/` 폴더에 저장

## 스프라이트 시트 공통 규격

- 캔버스: 1024 x 1536 px
- 그리드: 4열 x 3행 (프레임당 256 x 512 px)
- Row 0 (상단): 걷기/이동 모션 4프레임
- Row 1 (중단): 공격 모션 4프레임
- Row 2 (하단): 비워두기 (투명)
- 각 프레임 내 캐릭터는 중앙 하단 정렬
