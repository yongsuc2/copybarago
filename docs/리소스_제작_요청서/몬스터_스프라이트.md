## 테마 1: 미생물 세계 (챕터 1~10)

### 1. 아메바 (amoeba)

| 항목 | 내용 |
|------|------|
| 파일명 | `amoeba.png` |
| 색상 | 반투명 연보라/연분홍 |
| 특징 | 물렁물렁한 불규칙 형태, 큰 둥근 눈 1개(핵처럼), 젤리 같은 반투명 몸, 의족(가짜 팔) 2~3개 |

```
Create a 1024x1536 pixel sprite sheet of a cute chibi amoeba character in the same art style as the reference orange cat character. The amoeba should be translucent light purple/pink with an irregular blobby shape, one big round eye (like a nucleus), jelly-like semi-transparent body, and 2-3 pseudopods (fake arms). Thick black outlines, flat cel-shading, transparent background, 45-degree side view facing right.

Grid: 4 columns x 3 rows (each cell 256x512px).
- Row 0 (top): 4 idle animation frames (subtle breathing/swaying motion)
- Row 1 (middle): 4 attack animation frames (extending a pseudopod forward to engulf)
- Row 2 (bottom): empty/transparent

Character should be centered at the bottom of each cell.
```
