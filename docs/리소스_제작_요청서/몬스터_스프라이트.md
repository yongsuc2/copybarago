## 일반 몬스터 (5종)

### 1. 일개미 (ant)

| 항목 | 내용 |
|------|------|
| 파일명 | `ant.png` |
| 색상 | 갈색/진갈색 |
| 특징 | 큰 둥근 머리, 큰 눈, 더듬이 2개, 작은 둥근 몸, 짧은 다리 4개 (치비용 간소화) |

```
Create a 1024x1536 pixel sprite sheet of a cute chibi ant character in the same art style as the reference orange cat character. The ant should be brown/dark brown with a large round head, big cute eyes, two antennae, small round body, and 4 short legs (simplified from 6 for chibi style). Thick black outlines, flat cel-shading, transparent background, 45-degree side view facing right.

Grid: 4 columns x 3 rows (each cell 256x512px).
- Row 0 (top): 4 walking animation frames (legs moving cycle)
- Row 1 (middle): 4 attack animation frames (lunging/biting forward)
- Row 2 (bottom): empty/transparent

Character should be centered at the bottom of each cell.
```
